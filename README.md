# Gather

Gather is a native Android and Spring Boot application for creating small
hangouts and collecting guest RSVPs.

Organizers use the Android application to create and manage hangouts. Guests
open a public invite link in their browser, respond Yes, Maybe, or No, and can
add the hangout to their calendar without installing the app.

## Technology

### Android

- Kotlin
- Jetpack Compose
- Material 3
- MVVM
- StateFlow
- Hilt
- Retrofit

### Backend

- Kotlin
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Flyway

## Repository Structure

- `android-app/` — Native Android organizer application
- `backend/` — Spring Boot REST API
- `docs/` — Architecture and project documentation
